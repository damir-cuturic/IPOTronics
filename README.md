# IPOTronics
Hier komt alle gebruikte code die in de loop van IPOPRO61 gebruikt is. 

## Inhoud

* Thermometer - Temperatuur weergeven op het LCD scherm

* Buzzer Toggle - Melodie afspelen wanneer er op een knop gedrukt wordt

* Motor_Starter - Kijken hoe het motorshield werkt en welke code nodig is

* Kleurenwissel - Met behulp van een potentiometer alle kleuren op het scherm vertonen

* PingPongBang - Eindopdracht een pingpong ballen machine met behulp van de arduino aangestuurd
